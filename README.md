# 🌟 Site da Dra. Laudenice Lucena

Website profissional da Dra. Laudenice Lucena, Cirurgiã-Dentista em João Pessoa, PB. Criado para transmitir confiança, tecnologia e cuidado personalizado com foco em agendamentos rápidos e experiência do paciente.

---

## 🛠️ Tecnologias Utilizadas

- [Vite](https://vitejs.dev/) – Build super rápido para React.
- [React](https://reactjs.org/) – Biblioteca principal de UI.
- [TypeScript](https://www.typescriptlang.org/) – Tipagem estática para segurança e produtividade.
- [Tailwind CSS](https://tailwindcss.com/) – Estilização moderna e responsiva.
- [shadcn/ui](https://ui.shadcn.dev/) – Componentes elegantes e reutilizáveis.
